CareerView AI 🚀

CareerView AI is a modern AI-powered resume analyzer built using React, TypeScript, Vite, and Google Gemini API.

The application helps users analyze resumes, identify missing skills, improve ATS compatibility, and receive AI-generated career insights for specific job roles.

✨ Features

🔍 AI Resume Analysis

ATS-style resume evaluation
AI-generated improvement suggestions
Skill gap detection
Resume strengths & weaknesses analysis
Job-role-specific resume feedback
⚡ Fast Resume Input Modes

1. Paste Resume Text (Primary Mode)

Instant AI analysis
Optimized for speed and responsiveness
Recommended mode for best performance

🚀 Getting Started

Prerequisites

Make sure you have installed:

Node.js
npm
📦 Installation

Clone the repository:

git clone https://github.com/Tejusv7/CAREER-AI.git
Move into the project directory:

cd CAREER-AI
Install dependencies:

npm install
🔑 Environment Variables

Create a .env.local file in the root directory and add:

VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
▶️ Running the Project

Start the development server:

npm run dev
🏗️ Build for Production

npm run build
🚀 Deployment

This project is deployed using Vercel.
